# dovas.net

hello

My personal website and blog index. It is intentionally simple: static HTML, CSS, and a tiny bit of JavaScript for small browser niceties.

## Stack

- HTML
- CSS
- Vanilla JavaScript

## Features

- Personal homepage
- Blog/notes section
- Responsive layout
- No build step

## Development

```
open index.html
```

Yes, really. There's no build step. I will not apologize.

## Deployment

Hosted on GitHub Pages. DNS points to dovas.net.

If the site is down, it's probably a DNS issue. It's always a DNS issue.
